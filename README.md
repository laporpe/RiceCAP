# RiceCAP
Scripts associated with the manuscript: "Multi-environment evaluation and genomic prediction of agronomic traits in the southern US rice genepool"

Preprint DOI: 10.22541/essoar.175587849.99523492/v1
Available soon at The Plant Genome

If you need access to a script that is not included here, please contact the authors and it will be shared.

## Script Descriptions

- statgenGxE_FIXED_mixed_model_code.R: This file includes the modifications to the statgenGxE functions that were used to create the GxE figure (Figure 4 in The Plant Genome version of the manuscript).
- Two_State_statgenGxE_FIXED_mixed_model_code.R: This file includes the modifications to the statgenGxE functions that were used for the 2-state GxE analysis (in the supplemental materials).
